h1. .redspect


Application inspection tool for .net. It is permissive, provides helper methods to open a communication port from client. You can create your custom commands through compiled C#, or direct scripting through Roslyn and Ruby.

h2. .version 

2.0-edge-beta

h2. .features

* Helpers for hosting and connecting, using WCF named pipes.
* Support for compiled C#, Roslyn and IronRuby for command extensibility

h2. .usage

hosting

bc. Probe.Start() //starts server
Probe.Stop() //stops server

creating the service

bc. ProbeClient.Create()
